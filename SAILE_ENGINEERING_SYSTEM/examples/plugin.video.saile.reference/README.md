# SAILE Reference Video Plugin

Exemplo estrutural mínimo para estudo das skills. Ele não fornece catálogo nem conteúdo. Para testar playback, informe nas configurações uma URL MP4 que você tenha autorização para reproduzir.

## O que demonstra

- entry point fino;
- roteamento por allowlist;
- URL `plugin://` parametrizada;
- listagem e `endOfDirectory`;
- item reproduzível e `setResolvedUrl`;
- metadata com fallback de compatibilidade;
- settings moderno;
- SQLite no profile do add-on;
- SQL parametrizado;
- logging centralizado.

## O que não é

Não é um produto final, provider real, cliente de serviço ou base para contornar restrições de conteúdo.
