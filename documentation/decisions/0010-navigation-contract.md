# ADR 0010 — Ordem fixa de navegação

**Status:** aceito

## Decisão

sFy: Buscar, Minhas Playlists, Sincronizar Dados. sTv home: TV ao Vivo, VOD, Séries, Sincronizar Dados. Dentro de cada seção sTv: Buscar, Favoritos, depois conteúdo dinâmico.

## Consequências

A ordem é funcional, não estética, e possui teste automatizado.
