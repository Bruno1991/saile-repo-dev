# Kodi Testing Guide

## Smoke mínimo

- instalar o repository add-on;
- instalar/atualizar o add-on;
- abrir menu principal;
- abrir settings;
- navegar em listas vazias e preenchidas;
- cancelar sincronização;
- reproduzir fonte de teste autorizada;
- reiniciar Kodi e verificar banco/cache;
- inspecionar log por exceptions e segredos.

Registrar plataforma, versão, commit, passos e resultado.
