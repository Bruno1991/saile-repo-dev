# Developer Quickstart

1. copie este pacote para a raiz do SAILE;
2. preserve código e `.env` existentes fora do commit;
3. crie ambiente Python de desenvolvimento;
4. instale `requirements-dev.txt`;
5. execute os validadores;
6. rode `project_health.py` e atualize o status;
7. configure stubs Kodi apenas no ambiente de desenvolvimento;
8. faça mudanças em branch e valide o ZIP no Kodi real.
