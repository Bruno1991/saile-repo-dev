# Data Retention and Privacy

- favoritos e perfis permanecem até remoção do usuário;
- progresso pode possuir política configurável;
- cache expira e é limpo por orçamento de armazenamento;
- logs rotacionam e não contêm segredos;
- diagnósticos exportados passam por redaction;
- dados de provider não são enviados ao projeto SAILE;
- exclusão de perfil remove ou reassocia dados conforme confirmação explícita.
