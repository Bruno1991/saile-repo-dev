# Document Update Matrix

| Mudança | Atualizações obrigatórias |
|---|---|
| Nova rota ou parâmetro | Routing and Navigation, testes, traceability, project status |
| Mudança de playback | Playback Pipeline, Compatibility Matrix, testes de integração |
| Nova tabela/coluna/índice | Database Schema, Data Dictionary, Migration Policy, testes |
| Novo provider | Provider Contract, especificação, threat model, contrato de testes |
| Novo setting | settings XML, localização, secrets policy, documentação do usuário |
| Nova dependência | Dependency Policy, supply chain, licenças, performance budget |
| Correção de bug | teste de regressão, changelog, project status |
| Mudança de arquitetura | ADR, System Architecture, Module Boundaries, roadmap |
| Release | changelog, status, checklist, matriz de compatibilidade, hashes |
| Incidente de segurança | incident runbook, changelog Security, rotação e teste |
| Remoção/depreciação | Deprecation Policy, changelog, migração e comunicação |

O agente deve mencionar explicitamente quais linhas da matriz foram acionadas.
