# Supply Chain Security

- dependências mínimas e versionadas;
- origem e licença registradas;
- actions de CI com permissões mínimas;
- build em checkout limpo;
- hashes dos artefatos;
- revisão de alterações em workflows;
- sem download e execução de script remoto não verificado;
- assets e binários opcionais com checksum e matriz por plataforma;
- inventário do conteúdo do ZIP antes do release.
