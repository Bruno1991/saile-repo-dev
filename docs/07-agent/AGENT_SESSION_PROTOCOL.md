# Agent Session Protocol

## Entrada

Ler contexto, árvore, status e arquivos diretamente relacionados. Confirmar fatos no repositório antes de usar memória histórica.

## Durante

Manter escopo, anunciar riscos importantes, não editar segredo, executar validações incrementais e preservar alterações do usuário.

## Saída

Atualizar status, listar arquivos e testes, declarar o que não foi executado e definir próximo passo exato sem prometer trabalho assíncrono.
