# Prompt para iniciar o agente

```text
Leia integralmente o AGENTS.md da raiz e siga a ordem de leitura indicada nele.

Antes de modificar código:
1. audite os cinco add-ons ativos;
2. diferencie scaffold de implementação funcional;
3. confira os contratos de navegação e artwork;
4. apresente plano por arquivo, riscos, testes e rollback;
5. não leia, imprima ou publique valores do .env;
6. não crie produção dentro de .agents;
7. não reorganize IDs, menus ou fronteiras sem ADR.

Comece pelo MVP do plugin.video.stv. Não implemente sincronização LAN automática e não crie script.module.saile.ytdlp antes da prova técnica documentada.
```
