# Política de cache

Valores iniciais são centralizados e ajustados por evidência:

| Dado | TTL inicial |
|---|---:|
| categorias Xtream | 12 horas |
| canais | 1 hora |
| VOD e séries | 6 horas |
| metadados TMDB | 7 dias |
| busca musical | 15 minutos |
| URL de reprodução | nunca persistir |
| artwork remoto | cache do Kodi |

Expiração não apaga favoritos, progresso ou playlists. Falha remota pode usar cache vencido com indicação de modo degradado.
